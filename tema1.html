Introuge og starten på mutimedieproduktion



De første en til to uger på studiet, blev brugt på intro til skolen, samt intro til studiet. Det var overvældende, at lære så mange nye software, og termer, på så kort tid. Samtidig med vi vel og mærke skulle vænne os til et nyt sted, og lære nye mennesker, samt vores undervisere at kende.

Foruden alle formaliteter på campus, startede vi ved starten. Intro til HTML & CSS og vores første aflevering, var de første ting vi gik igang med. Afleveringen bestod af en simpel test, på om vi havde fået styr på at upoade til Github.

VSStudioCode, HTML, CSS & Github


VSStudioCode.

Er en software som kan benyttes af de fleste styresystemer, til at bygge og redigere kode. Man skal ikke undervurdere værdien af de extensions, altså udvidelser, man kan tilføje til softwaren.

Nogle af de extension vi har benyttet os af i undervisningen er:

  • Prettier
  • Live server
  • Samt farvetemaer


    • HTML (HyperText Markup Language).

    Er et standard kode sprog til at skabe hjemmesider, og som kan benyttes i VS-code. Det er altså fundamentet for strukturen. HTML benyttes til at fortælle browseren, hvordan den skal placere forskellige elementer efter eget ønske.

    Derefter kan man ændre de enkelte elementer. En box kan f.eks. få en ramme, tekst kan få farve, typografien kan ændres og meget mere. Dette gøres med CSS og de rette attributter.


    CSS (Cascading Style Sheet).

    Benyttes altså til at style vores HTML. Med styling af html kode kan der skabes nogen effekt på en side. I tema 1 blev vi introduceret for begge kodesprog, samt vi fik udleveret øvelser.


    Github

    Er en internetbaseret platform, designet til udvikling af software. Udviklere kan benytte platformen til at, opbevare projekter, dele projekter og følge med forandringer over tid. Ikke destod mindre kan udviklere arbejde på tværs af hinanden på større eller mindre projekter.
